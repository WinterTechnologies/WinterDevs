# Inkscape
## About
Inkscape is a free and open-source vector graphics editor used to create vector images, primarily in the Scalable Vector Graphics format.
Other formats can be imported and exported. Inkscape can render primitive vector shapes and text.

## Installation
- Download and run the installer from [inkscape.org](https://www.inkscape.org)
- Open Inkscape
